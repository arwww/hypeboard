# Changelog

All notable changes to Hypeboard are documented here.

## 1.0.0 - 2026-07-19

- Added the first productive static-dashboard architecture.
- Added a 30-stock configurable US universe.
- Added Stooq, Wikimedia Pageviews and FINRA short-sale-volume adapters.
- Added optional public social-feed adapter with explicit unavailable state.
- Added robust normalization, versioned score formulas and confidence scoring.
- Added deterministic score drivers and source-level freshness metadata.
- Added React dashboard, detail pages, methodology page, watchlist and filters.
- Added Python, JSON-schema and frontend component tests.
- Added daily GitHub Actions update and GitHub Pages deployment.
